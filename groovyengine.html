GROOVY ENGINE - PROJECT DESCRIPTION


1 / 3
2 / 3
3 / 3

Groovy Engine is a custom framework developed using the Godot game engine as a tool for me and my studio. This framework which I have named "Groovy Engine" is a jumping off point for making new game projects, something to assist my specific workflow and needs when making a game. Creating this significantly speeds up the development process of new games as it contains all the basic features that almost every game needs pre-made for the specific way that I use these techniques. Groovy Engine is the template from which all Godot-based games made by New Groove Studios will utilise in our production pipeline.

This is not available publicly as I want to keep it an in-studio product.
